KnightPawn
==========

A Project for My CSC340 AI class


8 Defined Moves for the Knight:

NE: X + 1, y + 2

EN: x + 2, y + 1

ES: x + 2, y - 1

SE: x + 1, y - 2

SW: x - 1, y - 2

WS: x - 2, y - 1

WN: x - 2, y + 1

NW: x - 1, y + 2




                               NW-------------NE
                             
                            WN                     EN
                        
                            WS                    ES
                        
                                 SW           SE
